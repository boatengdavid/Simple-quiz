# Simple Quiz App

This is a basic quiz application built as a learning project using HTML, CSS, and JavaScript.

---

## 🌍 Live Demo

https://boatengdavid.github.io/Simple-quiz/

---

## 📌 Description

The app displays multiple-choice questions and allows users to select answers.
At the end of the quiz, the total score is shown.

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## ⚙️ How It Works

* Questions are stored in a JavaScript array
* Each question is displayed one at a time
* User selects an answer
* The app shows whether the answer is correct or not
* Final score is displayed after all questions
